# Tuberculosis Patient Dashboard — Data Analysis Project

##  Project Overview
This project analyzes Tuberculosis (TB) patient data to uncover health trends, treatment patterns, and risk factors affecting patient outcomes.
The dashboard provides an interactive visual summary of patient demographics, symptoms, treatment duration, drug resistance, and overall treatment outcomes across different states in Nigeria.
The goal is to support healthcare stakeholders in:
•	Monitoring TB spread
•	Evaluating treatment effectiveness
•	Identifying high-risk groups
•	Improving public health interventions
